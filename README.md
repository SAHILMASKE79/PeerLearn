Language        →  Kotlin

Architecture    →  MVVM
Authentication  →  Firebase Auth (Email + Google)
Database        →  Cloud Firestore
Storage         →  Firebase Storage
Animation       →  Compose Animations
Min SDK         →  API 25 (Android 7.1)
